# 二进制包管理（基于YUM）

## 安装 yum install 

范例：yum install sudo 安装sudo软件（自动从互联网下载，并解决依赖关系）

## 检测升级 yum check-update

范例：yum check-update sudo 

## 升级 yum update

范例：yum update sudo 升级sudo软件

## 查询软件包信息 yum list

范例：yum list 查询yum源上所有软件包 

## 软件包信息 yum info

范例：yum info sudo 查询sudo的软件包信息

## 卸载 yum remove 

范例：yum remove sudo 删除sudo软件包
